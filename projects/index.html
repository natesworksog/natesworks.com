SwiftOS

What is SwiftOS

SwiftOS is an arch-based distrubution with a user friendly installer, desktop interface and gui package manager.

How can I download SwiftOS?

You can download the latest version of SwiftOS (Beta 0.2 Hotfix) here or you can build it manually from source code with Archiso.

Whats new in 0.2 Hotfix?

1. Fixed connecting to the internet

2. Added firewall

When will SwiftOS be released?

The first stable version of SwiftOS will most likely be released in September or November.
