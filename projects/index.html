SwiftOS

What is SwiftOS

SwiftOS is an arch-based distrubution with a user friendly installer, desktop interface and gui package manager.

How can I download SwiftOS?

You can download the latest version of SwiftOS (Beta 0.1.1) here or you can build it manually from source code with Archiso.

Whats new in 0.1.1?

It's whats 0.1 was supposed to be. I had my build script misconfigured and I tested a diffrent iso than I uploaded. This version is the first (working) version of SwiftOS!

When will SwiftOS be released?

The first stable version of SwiftOS will most likely be released in September or November.
